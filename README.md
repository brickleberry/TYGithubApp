## GitHub App (GitPeep)开发

### 概述
项目使用了ReactiveCocoa框架，实现了MVVM架构，分离视图(View)和模型(Model)。

#### MVVM四大优点：
- 低耦合
- 可重用性
- 独立开发
- 可测试

### 使用
请使用终端`cd`到`Podfile`所在的文件夹中

```
  1. pod repo update : 更新本地仓库 
  2. pod install : 下载安装
```
